    Teacher Management Application

    • Java SE |
    • MySQL |
    • jUnit

    This is a desktop application that helps a teacher manage student assessment scores and attendance. This project used a TDD approach where i created models. For each model i created test cases to makes sure that the behavior is correct. Working this way helped me confirm whether or not my implementation worked as expected. More especially with connection to the database. The project currently does not have any UX as yet because i wanted to make sure that the backend works fine before introducing frontend implementation. This helps me focus on the problem at a time instead of troubleshooting both frontend implementation and well as backend implementation.
